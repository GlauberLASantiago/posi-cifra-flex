📖 Como usar o app & Lista de acordes suportados

📖 Mini Tutorial

  1. Importe a partitura: Carregue um arquivo no formato .xml ou .musicxml (exportado do MuseScore, Finale, Sibelius, etc.) usando o botão de upload.
  2. Veja os diagramas: O aplicativo desenha automaticamente os diagramas de acordes (posições de violão) no grid de compassos.
  3. Selecione posições alternativas: Clique no botão "Mudar" sob qualquer acorde para abrir a galeria com todas as posições encontradas no braço do violão. Clique na opção desejada para atualizar a cifra e o diagrama.
  4. Ouça o acorde: Clique no diagrama do acorde para ouvir um arpejo dedilhado em áudio sintetizado de Violão Nylon de alta fidelidade.
  5. Filtre inversões e posições: Refine as posições configurando os filtros de limite de inversão, uso de cordas soltas, registro de altura (grave, médio ou agudo) e a abertura máxima de casas (3 ou 4 casas).
  6. Exporte para PDF: Clique em "Exportar PDF" para gerar um arquivo PDF otimizado em formato A4 Retrato com quebras de página limpas.

🎸 Acordes Suportados (Biblioteca Ampliada)

O Posi-Cifra-Flex suporta uma ampla gama de cifras, mapeando automaticamente do MusicXML e utilizando a biblioteca Tonal.js. Em acordes estendidos complexos (5+ vozes), o aplicativo realiza omit do 5º grau, 11ª ou 9ª para viabilizar posições tocáveis no violão, além de realizar simplificações inteligentes e suportar baixos alterados (Ex: D/F#).

Tríades

  • Maior (ex: C)
  • Menor (ex: Cm)
  • Aumentado (ex: Caug / C+)
  • Diminuto (ex: Cdim)
  • Power Chord / 5ª (ex: C5)

Suspensos

  • Suspenso 2 (ex: Csus2)
  • Suspenso 4 (ex: Csus4)
  • Sétima c/ 4ª suspensa (ex: C7sus4)

Tétrades (Sétimas)

  • Dominante (ex: C7)
  • Sétima Maior (ex: C7M / Cmaj7 / CM7)
  • Sétima Menor (ex: Cm7)
  • Sétima Diminuta (ex: Cdim7 / Co7)
  • Meio-Diminuto (ex: Cm7b5 / Cø)
  • Menor c/ Sétima Maior (ex: CmMaj7)
  • Sexta Maior / Menor (ex: C6 / Cm6)

Extensões e Alterados

  • Nona (ex: C9 / Cmaj9 / Cm9)
  • Décima Primeira (ex: C11 / Cm11)
  • Décima Terceira (ex: C13 / Cmaj13)
  • Sétima c/ Quinta Alterada (ex: C7#5 / C7b5)
  • Sétima c/ Nona Alterada (ex: C7#9 / C7b9)
  • Nona c/ Quinta Alterada (ex: C9#5 / C9b5)

* Nota: Cifras não listadas ou customizadas do arquivo MusicXML são automaticamente simplificadas para a tríade ou tétrade correspondente mais próxima, garantindo que diagramas válidos sempre sejam exibidos.
